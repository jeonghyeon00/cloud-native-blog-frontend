# Docker Project
***
Docker Project With Nginx, Spring Boot, React, MySQL

Automated Docker Image build and push by bash scripts.

## Project Features

---
- Board CRUD
- Authorization with JWT Token
- About Me Page
- OpenAPI Stocks Information
- TodoList

## Setup

---
```
    docker-compose up
```
then open localhost:80
